## Module 3: Libraries, APIs, Datasets and Models

### Python Libraries
- **pandas:** Data manipulation
- **NumPy:** Numerical computing
- **matplotlib/seaborn:** Visualization
- **scikit-learn:** Machine learning

- Application Programming Interfaces

- Examples: Twitter API, financial data APIs

### Datasets
- Sources: Kaggle, UCI, government data
- Formats: CSV, JSON, Excel, SQL databases

### Pre-trained Models

## Module 4: Jupyter Notebooks and JupyterLab

### Jupyter Notebook
- Interactive coding environment
- Mix code, text, visualizations
- Browser-based
- File format: `.ipynb`

### JupyterLab
- Next-generation interface
- More features (tabs, file browser)
- Preferred over classic Notebook

### Why Essential
- Industry standard for data science
- Great for exploration and learning
- Easy to share work
